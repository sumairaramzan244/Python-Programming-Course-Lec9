## Lecture 9 and 10  — Lists & Conditional Statements in Python  
In Lecture  9 and 10, I explored one of the most important data structures in Python — the List. A List is an ordered and changeable collection that allows duplicate members. I practiced adding, inserting, removing and managing list items using built-in methods. ---

## What I Learned  

**Lists:**  
* A List is an ordered and changeable collection written in square brackets `[ ]`
* Lists allow duplicate values and can store different data types together
* `append()` method adds a new item at the end of the list
* `insert(index, item)` method adds a new item at a specific index position in the list
* `remove(item)` method removes the first occurrence of a specific item from the list
* `pop(index)` method removes the item at a given index and also returns that removed item
* `clear()` method removes all items from the list and makes it empty
* `len()` function returns the total number of items currently in the list  

**Conditional Statements:**  
* Conditional statements allow a program to make decisions based on given conditions
* `if` statement executes a block of code only when the condition is True
* `elif` is used to check another condition when the previous `if` condition is False
* `else` block runs when none of the above conditions are True
* `and` operator is used to combine two or more conditions together in a single statement
* Conditions are evaluated from top to bottom — the first True condition runs and rest are skipped
